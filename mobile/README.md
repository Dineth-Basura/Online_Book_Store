# BookWorm Mobile App

## Run locally

1. Install dependencies

   ```bash
   npm install
   ```

2. Start Expo

   ```bash
   npm run start
   ```

3. Open on Android/iOS emulator, web, or Expo Go

## Run on a real phone

Set your backend URL before starting Expo:

```bash
set EXPO_PUBLIC_API_URL=http://YOUR_PC_LAN_IP:3000
npm run start
```

Example:

```bash
set EXPO_PUBLIC_API_URL=http://192.168.1.25:3000
npm run start
```

Make sure your phone and computer are on the same Wi-Fi network and backend is running.
